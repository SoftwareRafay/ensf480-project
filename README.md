# Instructions

### To Compile the Files:
`javac -d bin -cp "lib/\*" src/boundary/\*.java src/control/\*.java src/database/\*.java src/entity/\*.java`

### To Run the Program:
`java -cp "bin;lib/\*" Control.GUIController`

### For Mac:
#### Compile:
`javac -d bin -cp "lib/\*" src/Boundary/\*.java src/Control/\*.java src/Database/\*.java src/Entity/\*.java`

#### Run:
`java -cp "bin:lib/\*" Control.GUIController`
